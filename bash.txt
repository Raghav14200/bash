echo $SHELL -> this command is used to know which shell is currently being used.
ls ->this command is used to list files in that directory.
pwd ->this command is used to know present working directory.
cd ->this command is used to change directory.  
cat->is used to view the file or combine more than one file.
mkdir->this command is used to create directory.
mkdir -p ->this command is used to create directories,it creates it even if it does not exist.
head -> this command prints first 10 lines of file
head -n 20 ->this command prints first 20 lines of file
| -> this is used to feed standard output of one program to input of another program
cp -> this command is used to copy files
mv -> this command is used to rename or move files
wc -> this command is used to count no of lines,word and bytes
wc -l ->this command is used to count no of lines in file
wc -c -> this command counts no of bytes in file
wc -w -> this command counts no of words in file
curl -s ->this command loads ths http file and displays its content
tail -> this command prints last 10 lines of file
date -> this command is used to print date
date +'%Y'->this command displays the current year
date +'%m'->this command displays the current month
date +'%d'->this command displays the current date
