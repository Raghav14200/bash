echo $SHELL -> this command is used to know which shell is currently being used.
ls ->this command is used to list files in that directory.
pwd ->this command is used to know present working directory.
cd ->this command is used to change directory.  

cat->is used to view the file or combine more than one file.
mkdir->this command is used to create directory.
mkdir -p ->this command is used to create directories,it creates it even if it does not exist.
head -> this command prints first 10 lines of file
head -n 20 ->this command prints first 20 lines of file
